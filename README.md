# Ejercicios-Linux
Ejercicios de Linux

## Ejercicio1
-Show all the jpg pictures in the current directory.


```console
ls *.jpg
```

## Ejercicio2
-Display all the files in the directory /usr/bin starting with letter “j”.


```console
ls /usr/bin/j*
```

## Ejericio3
-Show all the files in the directory /usr/bin starting with the letter “k”, with an “a”
in the 3 rd place.

```console
ls /usr/bin/k?a*
```
